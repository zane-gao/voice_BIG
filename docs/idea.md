**项目是什么：**
计划开发一个“语音 + 草图”多模态科研方法图实时生成系统，用户可以一边手绘草图、一边口头描述思路，系统自动将其转化为结构化、规整化、论文风格的方法图。

**动机：**
科研早期想法往往以零散草图和口头表达的形式出现，但从“混乱想法”到“清晰方法图”需要大量整理成本，因此我们希望用语音信息处理与多模态技术降低科研表达门槛。

**市场需要与用户痛点：**
学生、科研人员和产品研发者经常需要把白板讨论、手写笔记和临时想法整理成汇报图、论文图或方案图，但现有工具依赖手动排版，效率低、结构化能力弱、难以捕捉口头语义。

**使用场景与流程：**
用户在组会、头脑风暴或论文构思时绘制草图并同时语音讲解说明，系统完成语音转写、草图识别、语义补全和结构化建模，最终生成可编辑的 Mermaid、SVG、PPT 或论文风格方法图（也可调用生图模型 api 直接生成）。

**验收标准：**
系统应能完成从“语音 + 草图输入”到“结构化方法图输出”的完整闭环，并通过 ASR 准确率、节点/边识别准确率、图结构还原度、生成耗时和用户主观评分等指标验证其有效性。
