# SketchVoice 项目说明

## 这个项目在做什么

这是一个浏览器里的“草图 + 语音”方法图生成工具。

你可以在前端画一个科研流程草图，补一段口述或上传音频，系统会把这些输入整理成结构化的知识图/方法图，并输出：

- 可编辑的 Mermaid 源码
- 图形预览
- 结构化 JSON
- 论文风格的 AI 方法图
- 带语音讲解的 WebM

从 `README.md` 看，它更像一个可交互的 MVP；从 `report/main.pdf` 看，它又被包装成一个研究原型，核心方法叫 `MMSB-Graph`。报告里的重点不是“聊天”，而是把草图观测、语音语义和目标知识图统一起来，验证语音能补节点、纠正边方向、降低人工修改成本。

一句话说，这个项目是在做“把手绘科研草图和口头说明，自动整理成规范方法图”的 Web 应用。

## 前端能实现的操作

### 输入

- 在左侧画布手绘草图
- 切换画笔、橡皮
- 清空画布
- 载入演示样例
- 录音
- 上传音频
- 播放样例音频
- 编辑转写文本
- 选择方法图方向 `LR` / `TD`

### 生成

- 点击“生成方法图”
- 自动获取语音转写结果
- 自动生成结构化图数据和 Mermaid
- 停止输入后自动生成草稿图
- 手动生成终稿图
- 选择终稿图风格
- 生成语音讲解
- 选择讲解音源和音色

### 查看与导出

- 在 Mermaid / 图形 / AI 图像 / JSON 之间切换
- 下载 Mermaid
- 下载 SVG
- 下载 JSON
- 下载草稿图
- 下载终稿图
- 播放讲解音频
- 下载讲解 WebM

## 你可以把它理解成什么

- 不是通用绘图软件
- 不是普通语音转文字工具
- 也不是纯文本生成器
- 它是一个面向科研方法图的多模态工作台

如果把论文视角和产品视角合在一起看，前端负责“输入草图和语音、展示中间结果、导出最终图和讲解”，后端负责“识别、结构化、生成图像和语音”。
